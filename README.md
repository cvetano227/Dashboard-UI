# Weather Dashboard UI

This project is a simple **Weather Dashboard UI** built using **HTML and CSS**.  

## 🚀 Features
- Static weather dashboard layout
- Responsive flexbox-based structure
- Custom typography using Google Fonts
- Icons from Bootstrap Icons
- Styled components (location, time, weather info, forecast)

## 🛠️ Tech Used
- HTML5
- CSS3 (Flexbox)
- Bootstrap Icons CDN
- Google Fonts (Work Sans)

## 📌 Purpose
The goal of this project was to improve:
- Layout structuring
- CSS positioning and spacing
- Converting UI designs into code

## 📷 Preview
A weather dashboard showing:
- Location & time
- Current weather conditions
- 5-day forecast

---

*This is a static UI project (no JavaScript or API integration).*
